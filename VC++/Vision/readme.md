 # 视觉软件demo

## 使用opencv3.4.12
 
## 导电柱厚度
![导电柱.jpg](https://i.loli.net/2021/09/29/2fK1bEqHpA3Lxc4.jpg =500x)

### 使用方法

* 选择导电柱图像在图像上拖画两个矩形框如下图

![图1.jpg](https://i.loli.net/2021/09/29/WZfhxHnPu9MyoCE.jpg =500x)

* 设置直线搜索方式（向上or向下）及搜索点数
* 点击测试

### 目的

* 判断导电柱厚度，及是否均匀
